Projekt Parallax Scrolling
=====

### Repository Richtlinien und Informationen:
* Hinsichtlich Code-Formatierung bitte unsere konventionen einhalten.
* Immer nur im Branch *general-devel* pushen!
* Branches *master* und *general-devel* laufen parallel.
* Möglichst anhand der Issues committen. Nicht vergessen sich selbst einem Issue zuzuweisen, bevor mit der Arbeit begonnen wird! Auch nicht vergessen das Issue hinterher wieder zu schließen.
* Milestones / releasefertige Commits werden regelmäßig in *master* übernommen.
* Neue Branches niemals von *master* aus!
